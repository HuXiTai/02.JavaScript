1 - 内置对象

1.1 内置对象

JavaScript 中的对象分为3种：自定义对象、内置对象、浏览器对象。

前面两种对象是 JS 的基础内容，属于 ECMAScript；第三个浏览器对象属于 JS 独有的，JS API讲解内置对象就是指 JS 语言自带的一些对象，这些对象供开发者使用，并提供了一些常用的或是最基本而必要的功能（属性和方法），内置对象最大的优点就是帮助我们快速开发。

JavaScript 提供了多个内置对象：JSON、Math、 Date 、Array、String等

1.2 JSON对象

JavaScript Object Notation (JSON) 是一种数据传输格式，本质上是一个字符串。

JSON对象包含两个方法:

方法名 作用
JSON.parse 将json字符串转成js对象
JSON.stringify 将js对象转成json字符串

1.3 Math对象

​ Math 对象不是构造函数，它具有数学常数和函数的属性和方法。跟数学相关的运算（求绝对值，取整、最大值等）可以使用 Math 中的成员。

属性、方法名 功能
Math.PI 圆周率
Math.ceil() 向上取整
Math.floor() 向下取整
Math.round() 四舍五入版 就近取整 注意 -3.5 结果是 -3
Math.abs() 绝对值
Math.max()/Math.min() 求最大和最小值
Math.random() 获取范围在[0,1)内的随机值
Math.pow() 幂运算
Math.sin() 正弦运算

​ 注意：上面的方法使用时必须带括号

​ 获取验证码：

function getRandomCode(n){
    //这个函数是用来获取验证码的
    var str = '1234567890';
    var code = '';
    for(var i = 0; i < n; i++){
        code += str[Math.floor(Math.random()*str.length)];
    }
    return code
}

​ 获取指定范围内的随机整数：

function getRandom(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min; 
}

1.5 包装对象

为了方便操作基本数据类型，JavaScript 还提供了三个特殊的引用类型：String、Number和 Boolean。基本包装类型就是把简单数据类型包装成为复杂数据类型，这样基本数据类型就有了属性和方法。

包装对象一共有三种：数字、字符串、布尔值

var a = 100;
console.log(a); // 100，Number类型
console.log(a instanceof Object); // false
console.log(a.toString());// 100
console.log(a instanceof Object);// false

var a = new Number(100);// 创建的是Number对象，对象类型
console.log(a instanceof Object); //true

var und;
console.log(und.toString());//报错，原因是因为undefined没有包装对象

a.toString执行到这一行的时候，其实干了很多事： 1、a = new Number(a); 先把数字基本值转化为包装对象 2、a.toString(); 调用的其实是包装对象的原型当中的toString方法 3、调用结束后自动再让a变回基本值 a = 100;

同理：

// 下面代码有什么问题？
var str = 'andy';
console.log(str.length);

按道理基本数据类型是没有属性和方法的，而对象才有属性和方法，但上面代码却可以执行，这是因为js 会把基本数据类型包装为复杂数据类型，其执行过程如下 ：

// 1. 生成临时变量，把简单类型包装为复杂数据类型
var str = new String('andy');
// 2. str对象访问length属性并打印
console.log(str.length);
// 3. 将字符串赋值给变量
str = "andy";

1.6 字符串对象

字符串的不可变

​ 指的是里面的值不可变，虽然看上去可以改变内容，但其实是地址变了，内存中新开辟了一个内存空间。

当重新给字符串变量赋值的时候，变量之前保存的字符串不会被修改，依然在内存中重新给字符串赋值，会重新在内存中开辟空间，这个特点就是字符串的不可变。由于字符串的不可变，在大量拼接字符串的时候会有效率问题。

根据字符返回位置

​ 字符串通过基本包装类型可以调用部分方法来操作字符串，以下是返回指定字符的位置的方法：

方法名 说明
s.indexOf(s1, [index]) 从前往后查找子字符串s1并返回位置，可选参数index表示指定从哪开始找，默认从0开始，没找到返回-1。
s.lastIndexOf(str, [index]) 从后往前找，只找第一个匹配的，可以指定从index开始向前搜索，没有出现，则该方法返回 -1。

​ 案例：查找字符串"abcoefoxyozzopp"中所有o出现的位置以及次数

  1. 先查找第一个o出现的位置
  2. 然后 只要indexOf 返回的结果不是 -1 就继续往后查找
  3. 因为indexOf 只能查找到第一个，所以后面的查找，利用第二个参数，当前索引加1，从而继续查找

str[index] 获取指定位置的字符，html5新增，和charAt()等效

字符串操作方法

​ 字符串通过基本包装类型可以调用 部分方法来操作字符串，以下是ES5的操作方法：

方法名 说明
s.concat(s1, s2, ...) 字符串拼接，返回新的字符串，等效于+
String.fromCharCode(unicode,...) 返回给定的一个或多个unicode编码表示的字符组成的字符串
s.replace(regexp/s1, s2) s1表示被替换的字符串/正则匹配的字符串，s2表示要替换为的字符串，返回的是新的字符串。
s.localeCompare(s1) 按位比较s1和s两个字符串的大小。s小于s1，返回小于0的数；s大于s1返回大于0的数；相等返回0。
s.slice(start, [end]) 返回从 start 开始（包括 start）到 end 结束（不包括 end）为止的所有字符组成的一个新的字符串，-1表示最后一个字符。
s.split(separator, [howmany]) 把一个字符串分割成字符串数组，数组的长度小于等于howmany，separator指定空串会把每一个字符都切割。
s.substr(start, length) start表示从哪开始截取，length表示截取的长度，返回新字符串。
s.substring(start, end) 截取字符串，可以自动根据位置大小决定起始和结束位置，功能和slice相同，但是它不接受负值。
s.toLowerCase() 用于把字符串转换为小写。
s.toUpperCase() 用于把字符串转换为大写。
s.valueOf() 如果对象有PrimitiveValue，就返回PrimitiveValue的值；没有则返回对象本身。只有包装对象才有PrimitiveValue。
s.toString() 把一个对象转换为字符串

包装对象调用valueOf和toString是不一样的：

  • 包装对象调用的valueOf和toString都是自己原型身上的。
  • 非包装对象调用的valueOf都是Object原型当中的，toString调用的是自己原型当中的。

1.7 数组对象

创建数组的两种方式

  • 字面量方式

    • 示例代码如下：

      var arr = [1,"test",true];

  • new Array()

    • 示例代码如下：

      var arr = new Array();

      ​ 注意：上面代码中arr创建出的是一个空数组，如果需要使用构造函数Array创建非空数组，可以在创建数组时传入参数

      ​ 参数传递规则如下：

      • 如果只传入一个参数，则参数规定了数组的长度

      • 如果传入了多个参数，则参数称为数组的元素

检测是否为数组

  • instanceof 运算符

    • instanceof 可以判断一个对象是否是某个构造函数的实例

      var arr = [1, 23];
var obj = {};
console.log(arr instanceof Array); // true
console.log(obj instanceof Array); // false

  • Array.isArray()

    • Array.isArray()用于判断一个对象是否为数组，isArray() 是ES6的方法

      var arr = [1, 23];
var obj = {};
console.log(Array.isArray(arr));   // true
console.log(Array.isArray(obj));   // false
