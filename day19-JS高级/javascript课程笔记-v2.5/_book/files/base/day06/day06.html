1. 立即调用函数表达式（IIFE）

立即调用函数表达式（Immediately Invoked Function Expression）是一个在定义时就会立即执行的JavaScript函数。

它的格式是：

(function () {
    statements
})();

或者

(function(){ 
    statements
}());

这是一个自执行的匿名函数，主要包含两部分。第一部分是包围在圆括号运算符( )里的词法作用域。这不仅避免了外界访问此 IIFE 中的变量，而且又不会污染全局作用域。第二部分再一次使用 () 创建了一个立即执行函数表达式，JavaScript 引擎到此将直接执行函数。

特点：

  • 函数定义的时候同时执行

  • 只执行一次

作用：

  • 防止内外部变量重名
  • 对项目的初始化（只需要执行一次的代码）
  • 隐藏内部代码，需要外部访问的可以对外暴露接口（了解即可，后面会详细学）

应用：

jQuery开篇用的就是立即执行函数，立即执行函数常用于第三方库，好处在于隔离作用域，任何一个第三方库都会存在大量的变量和函数，为了避免变量污染（命名冲突），开发者们想到的解决办法就是使用立即执行函数。

2. 回调函数

概念：一个函数A被作为参数传递给另一个函数B，在函数B里的某一时刻去调用函数A，那么A就被称为回调函数。

不带参数的回调函数：

function callback() {  // 这是一个回调函数
    alert("execute callback");
}

function execute(func) {
    func();
}
execute(callback);

带参数的回调函数

function callback(value) { // 这是一个回调函数
    alert(value);
}

function execute(func, value) {
    func(value);
}
execute(callback, 'hi,js');

3. 函数的递归调用

3.1什么是递归

递归：如果一个函数在内部可以调用其本身，那么这个函数就是递归函数。简单理解:函数内部自己调用自己, 这个函数就是递归函数

注意：递归函数的作用和循环效果一样，由于递归很容易发生“栈溢出”错误（stack overflow），所以必须要加退出条件return。

例如：我们要实现从0-3的打印，打印完3程序终止，我们可以用循环来实现：

function foo() {
    for (var i = 0; i <= 3; i++) {
        console.log(i);
    }
}
foo();

那么还有没有其他的办法呢？

function foo(i) {
    if (i === 3) {
        console.log(i);
        return;
    } else {
        console.log(i);
        foo(++i);
    }
}
foo(0);

执行过程：

3.2利用递归求1~n的阶乘

//利用递归函数求1~n的阶乘 1 * 2 * 3 * 4 * ..n
 function fn(n) {
     if (n == 1) { //结束条件
       return 1;
     }
     return n * fn(n - 1);
 }
 console.log(fn(3));

3.3利用递归求斐波那契数列

// 利用递归函数求斐波那契数列(兔子序列)  1、1、2、3、5、8、13、21...
// 用户输入一个数字 n 就可以求出 这个数字对应的兔子序列值
// 我们只需要知道用户输入的n 的前面两项(n-1 n-2)就可以计算出n 对应的序列值
function fb(n) {
  if (n === 1 || n === 2) {
        return 1;
  }
  return fb(n - 1) + fb(n - 2);
}
console.log(fb(3));

4. 对象

4.1 对象的相关概念

  • 什么是对象？

    在 JavaScript 中，对象是一组无序的相关属性和方法的集合，所有的事物都是对象，例如字符串、数值、数组、函数等。
对象是由属性和方法组成的。

    • 属性：事物的特征，在对象中用属性来表示（常用名词）

    • 方法：事物的行为，在对象中用方法来表示（常用动词）

  • 为什么需要对象？

    保存一个值时，可以使用变量，保存多个值（一组值）时，可以使用数组。

    如果要保存一个人的完整信息呢？

    例如，将“张无忌”的个人的信息保存在数组中的方式为：

    var arr = ['张三疯', '男', 128,154];

    上述例子中用数组保存数据的缺点是：数据只能通过索引值访问，开发者需要清晰的清除所有的数据的排行才能准确地获取数据，而当数据量庞大时，不可能做到记忆所有数据的索引值。

    为了让更好地存储一组数据，对象应运而生：对象中为每项数据设置了属性名称，可以访问数据更语义化，数据结构清晰，表意明显，方便开发者使用。

    使用对象记录上组数据为：

    // 键值对  键名：值 
var obj = {    
    name:"张三疯",    
    sex:"男",    
    age:128,    
    height:154
}

    JS中的对象表达结构更清晰，更强大。

4.2 创建对象的三种方式

1. 使用对象字面量创建对象：

  • 花括号 { } 里面包含了表达这个对象的属性和方法；

  • { } 里面采取键值对的形式表示，键值对之间用冒号（:）分隔；

  • 对象的属性（键），本质上都是字符串，只不过大部分属性是可以省略引号不写的，少数情况会出现不符合命名规范的,就必须带引号；

  • 对象的属性值可以是任意类型的值（数字类型、字符串类型、布尔类型，函数类型等），如果是函数，那么我们称这个属性叫做对象的方法；

    格式如下：

    /*
  var 对象名 = {
      key:value,
    key:value,
    key:value
  }
*/

    var star = {
    name : 'dilireba',
    'age' : 18,
    sex : '女',
    sayHi : function(){
        alert('大家好啊~');
    },
    'personality-type':'温柔', //这个属性名必须使用引号
};

    上述代码中 star即是创建的对象。

2. 利用内置构造函数创建对象

  • 创建空对象

    var andy = new Object();

    通过内置构造函数Object创建对象，此时andy变量已经保存了创建出来的空对象

  • 给空对象添加属性和方法

    • 通过对象操作属性和方法的方式，来为对象增加属性和方法

      示例代码如下：

      andy.name = 'dilireba';
andy.age = 18;
andy.sex = '女';
andy.sayHi = function(){
    alert('大家好啊~');
}

    注意：

    • Object() ：第一个字母大写
    • new Object() ：需要 new 关键字
    • 使用的格式：对象.属性 = 值;
3. 利用自定义构造函数创建对象

  • 构造函数

    • 构造函数：是一种特殊的函数，主要用来初始化对象，即为对象成员变量赋初始值，它总与 new 运算符一起使用。我们可以把对象中一些公共的属性和方法抽取出来，然后封装到这个函数里面。

    • 构造函数的封装格式：

      function 构造函数名(形参1,形参2,形参3) {
     this.属性名1 = 参数1;
     this.属性名2 = 参数2;
     this.属性名3 = 参数3;
     this.方法名 = 函数体;
}
// this是一个内置的局部变量,它存储的是对象的地址。

    • 构造函数的调用格式

      var obj = new 构造函数名(实参1，实参2，实参3)

      以上代码中，obj即接收到构造函数创建出来的对象。

    • 注意事项

      1. 构造函数约定首字母大写。
      2. 函数内的属性和方法前面需要添加 this ，表示当前对象。
      3. 构造函数中不需要 return 返回结果。
      4. 当我们创建对象的时候，必须用 new 来调用构造函数。

    • 其他

      构造函数，如 Stars()，抽象了对象的公共部分，封装到了函数里面，它泛指某一大类（class）。 创建对象，如 new Stars()，特指某一个，通过 new 关键字创建对象的过程我们也称为对象实例化。

  • new关键字的作用

    1. 在构造函数代码开始执行之前，创建一个空对象；
    2. 修改this的指向，把this指向创建出来的空对象；
    3. 执行函数的代码
    4. 在函数完成之后，返回this---即创建出来的对象

4.3 对象的操作

对象的增删改查

1. 查

  • 访问对象的属性

    • 对象里面的属性调用 : 对象.属性名 ，这个小点 . 就理解为“ 的 ”
    • 对象里面属性的另一种调用方式 : 对象['属性名']，注意方括号里面的属性必须加引号

    示例代码如下：

    console.log(star.name)     // 调用名字属性
或者
console.log(star['name'])  // 调用名字属性，有特殊字符的只能以这种方式
或者
var n = 'name'; 
console.log(star[n]) // 以变量存储的数据为属性取属性对应的值

    说明：如果找不到这个属性，那么这个属性打印出来的值是undefined，不会报错。

  • 调用对象的方法

    对象里面的方法调用：对象.方法名() ，注意这个方法名字后面一定加括号。

    示例代码如下：

    star.sayHi();// 调用 sayHi 方法,注意，一定不要忘记带后面的括号
或
star['sayHi']();
或
fn = "sayHi";
star[fn]();
2. 增、改

原则：有则更改，无则添加

star.hobby = '跳舞'; 
或者
star['hobby'] = '跳舞'; //当对象的属性包含不合法字符的时候必须用这种方式
或者
h = 'hobby';
star[h] = '跳舞';
3. 删

删除属性或方法

delete star.hobby;
或者
delete star['hobby'];
或者
h = "hobby";
delete star[h];

console.log(star['hobby']); // undefined

4.4 遍历对象

  • for...in 语句用于对对象的属性进行循环操作。 

    for (var k in obj) {
    console.log(k);      // 这里的 k 是属性名
    console.log(obj[k]); // 这里的 obj[k] 是属性值
}

    语法中的变量是自定义的，它需要符合命名规范，通常我们会将这个变量写为 k 或者 key。

  • for...in 语句用于对数组进行循环操作。

    var arr = [1,2,3,4];

arr.heihei = 'haha';//A.B 那么A一定是对象，B一定是对象的属性，把数组当对象用加了一个属性和值
arr['hi'] = 5;

console.log(arr);

//数组当数组用的使用，我们用for循环去遍历；for循环只能遍历出数组当中数字属性的值,数组的长度依然为4
for(var i = 0; i < arr.length; i++){
    console.log(arr[i]);
}
console.log(arr.heihei);
console.log(arr.hi);
// 如果想要用for循环将属性和值也遍历出来，我们可以用for...in遍历
for (var i in arr) {
    console.log(i);
    console.log("val:", arr[i]);
}

  • for...in 语句用于对函数进行循环操作。

    function fn(){
    console.log('call fn');
}

fn();//当函数去用，就是调用函数实现函数表面上的功能；一个东西加括号，那么括号前面一定是函数，函数是在当函数用；

//函数当对象用
fn.hehe = 'heihei';   //A.B  那么代表这个函数是在当对象用；
console.log(fn);
console.dir(fn);

for(var key in fn){
    console.log(key,fn[key]);
}

4.5 变量、属性、函数、方法总结

属性是对象的一部分，而变量不是对象的一部分，变量是单独存储数据的容器

  • 变量：单独声明赋值，单独存在

  • 属性：对象里面的变量称为属性，不需要声明，用来描述该对象的特征

    方法是对象的一部分，函数不是对象的一部分，函数是单独封装操作的容器

  • 函数：单独存在的，通过“函数名()”的方式就可以调用

  • 方法：对象里面的函数称为方法，方法不需要声明，使用“对象.方法名()”的方式就可以调用，方法用来描述该对象的行为和功能。
